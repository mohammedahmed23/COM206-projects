# COM206-VISUAL PROGRAMMING

Welcome to the **COM206-VISUAL PROGRAMMING** course repository.  
This repository is designed to guide students through the fundamentals of GUI design and programming across multiple platforms, culminating in a final project.

---

## 📖 Course Overview

This course introduces students to **Graphical User Interfaces (GUIs)** as both a design philosophy and a programming practice.  
Students will explore design principles, prototyping, and implementation across **mobile, desktop, and web platforms**, using tools such as **Figma, Flutter, PyQt, HTML/CSS/JS**.

---

## 🗓 Weekly Outline

### Week 1 – Foundations of GUI
- **Topics:** What is a GUI? CLI vs. GUI, usability, accessibility, event-driven programming.  
- **Activity:** Analyze famous apps/websites for good vs. poor UI design.  
- **Outcome:** Understand GUI as both design and programming.

### Week 2 – Prototyping with Figma
- **Topics:** Wireframes vs. prototypes, components, auto-layout, design systems.  
- **Activity:** Create a simple mobile app prototype (login + dashboard).  
- **Outcome:** Visualize GUI before coding.

### Week 3 – Mobile GUI with Flutter (Basics)
- **Topics:** Flutter architecture, widgets, state, hot reload, layouts (Row, Column, Container).  
- **Activity:** Build a basic mobile app with navigation and buttons.  
- **Outcome:** Connect design (Figma) to implementation.

### Week 4 – Mobile GUI with Flutter (Advanced)
- **Topics:** State management (setState, Provider), forms, validation, user input.  
- **Activity:** Extend app with a registration form.  
- **Outcome:** Handle dynamic user interaction.

### Week 5 – Desktop GUI with PyQt (Basics)
- **Topics:** PyQt architecture, widgets, signals/slots, layout managers.  
- **Activity:** Build a desktop app with menus and dialogs.  
- **Outcome:** Understand event-driven programming in Python.

### Week 6 – Desktop GUI with PyQt (Advanced)
- **Topics:** Custom widgets, styling with Qt Designer, file handling, persistence.  
- **Activity:** Create a note-taking app with save/load functionality.  
- **Outcome:** Integrate GUI with backend logic.

### Week 7 – Web GUI with HTML & CSS
- **Topics:** Structure with HTML, styling with CSS, responsive design (Flexbox, Grid).  
- **Activity:** Build a simple responsive portfolio site.  
- **Outcome:** Grasp layout and styling fundamentals.

### Week 8 – Web GUI with JavaScript
- **Topics:** DOM manipulation, events, forms, dynamic content.  
- **Activity:** Add interactivity (form validation, dynamic lists).  
- **Outcome:** Create interactive web apps.

### Week 9 – Integration & Final Project
- **Topics:** Comparing GUI platforms (mobile, desktop, web), usability best practices.  
- **Activity:** Choose one platform (Flutter, PyQt, or Web) and build a small project based on your Figma prototype.  
- **Outcome:** Demonstrate end-to-end GUI design and programming skills.

---

## 🎯 Teaching Flow

1. **Design Philosophy** → GUI is about structure, usability, and interaction.  
2. **Prototype in Figma** → Visualize before coding.  
3. **Implement Across Platforms** → Mobile (Flutter), Desktop (PyQt), Web (HTML/CSS/JS).  
4. **Final Integration Project** → Reinforce design-to-code pipeline.
