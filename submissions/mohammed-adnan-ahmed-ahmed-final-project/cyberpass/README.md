Project Title:

CyberPass: A desktop application for secure password generation.

Research Question / Problem Statement:

Weak passwords are a common cybersecurity issue. Many users create simple or repeated passwords that can easily be guessed or attacked.

Objective of the Project:

The objective of this project is to develop a Python desktop application using PySide6 that allows users to generate strong and secure passwords.

Expected Outcome:

A fully functional desktop application that:

Generates secure passwords
Allows custom password length
Supports symbol inclusion
Displays password strength
Provides a clean user interface


Approach to Problem Solving:

The application works in 3 main parts:

User Interface:

The GUI contains:

Password display field
Password length selector
Symbol checkbox
Generate button
Copy button
Backend:

The backend generates random passwords using Python libraries and evaluates password strength.


The generated password and its strength are displayed clearly for the user.

Application Flow (CyberPass App)
Start application
Display main window
User selects password settings
User clicks “Generate Password”
Application generates password
Display generated password and strength
Allow user to copy password
